Function-scoped or globally scoped.
Can be redeclared and updated.
Hoisted to the top of their scope (initialized as undefined).

Block-scoped (limited to the block {} they are defined in).

Block-scoped.

Object Destructuring with Rest and Spread:

const obj = { a: 1, b: 2, c: 3 };
const { a, ...rest } = obj;
const newObj = { ...rest, d: 4 };
console.log(newObj); // { b: 2, c: 3, d: 4 }

6. Limitations
Shallow Copying: Both spread and rest create shallow copies, so nested objects/arrays share references.
Object Spread/Rest Browser Support: Supported in modern browsers but may require polyfills for older ones (e.g., pre-2018 environments).
Rest in Functions: Only one rest parameter is allowed, and it must be the last parameter.
Spread with Non-Iterables: Spread only works with iterables or objects (e.g., ...123 throws a TypeError).
